Usage

Photo Search

- Enter your search keyword in the "Search for something" input field

- Click the "Search" button

- Search results will be displayed as cards

Clearing Results

- Click the "Clear" button to reset current search results

API Integration

Unsplash API

- The application performs photo searches using the Unsplash API

- Secure access is provided through Authorization header

- Search queries are made with GET requests
